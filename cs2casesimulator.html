What's the game about?

CS2 is a shooter game which contains a case opening system. I found it quite interesting and wanted to try and recreate the case opening system myself.

There are 10 different cases that can be opened with the same odds as CS2's case opening system. I made everything very simular to CS2 by making the UI, skins and cases look the same.

In this simulator you can choose which case you want to open and you can then buy keys for it. After unlocking the case you'll get a roll animation which will then end up on your winning skin.

How does it work?

Upon unlocking the case 50 weapon showcases will spawn, which each contain a random skin. Every weapon showcase will move towards the left and there is a winline in the middle, whatever skin lands on it will be the winning showcase. The closer the winning showcase is from the winline, the less speed it will have moving left. Once the winning showcase is on the winline, it will slowly stop moving. The winning showcase has already been determined from the moment you open the case, just like CS2's case openings / gambling in general.

I used scriptable objects and prefabs to create an easy way to change what case is selected and opened.
It's also very easy for me to add new cases and skins to the project because of the efficient use of scriptable objects.

What did I learn?

User Interface

I took the UI very seriously this project and wanted it to look amazing and work perfectly. My goal was to make the UI the same as CS2's layout and color scheme. This project has the best UI I've ever made before, it's easy to navigate and looks very good.

Everything I wanted to add initially

At the start of the project I planned out a lot of features to add. The following features I never ended up adding, but would be very cool additions to the game:

- Inventory system that saves what skins you've won in the skins inventory tab
- A level system which unlocks a new case each level, opening cases grants xp
- Daily special cases which unlock depending on your level
- Inventory saves with PlayerPrefs (skins, keys and level)
- You can select skins to bet in the upgrader. To upgrade a skin's value you can choose a percentage to hit, the lower the percentage is, the more the skin will be upgraded.
