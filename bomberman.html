What's the game about?

Bomberman is my biggest C++ project so far, since my only prior project with C++ is Bowborne.

The goal in Bomberman is to kill every enemy with your bombs and to find the exit door, which can only be used once every enemy has been defeated.

There is both a singleplayer and multiplayer mode so you can play face these enemies alone or with help from a friend.

Each stage gets progressively harder until you inevitably lose.
