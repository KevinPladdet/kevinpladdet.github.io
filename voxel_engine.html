Minecraft inspired Voxel Engine

Over the course of 8 weeks I built a Minecraft inspired voxel engine from scratch in C++ using OpenGL and GLFW.

The game features an infinite procedurally generated world split into chunks, containing caves, oceans and a dynamic day/night cycle.

You can explore the world by walking around in survival mode or by freely flying around in creative mode.

Place and destroy blocks to build anything you want, from a beautiful house to a challenging parkour.

The game includes background music, sound effects, a fully functional hotbar and smooth movement.

Lastly there is a debug menu (toggle with F3) created with ImGui, that allows you to enable or disable features and to tweak them to your liking in real time.
