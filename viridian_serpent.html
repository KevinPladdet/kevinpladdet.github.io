Ray Tracing Renderer with playable Snake game

Viridian Serpent is a voxel-based ray tracer built on the CPU, featuring a playable Snake game integrated into the renderer.

Rendering Features:
- Support for point lights, spot lights and area lights
- Progressive rendering with an accumulator to reduce noice gradually over time
- Reprojection: Blends previous frame with current frame to reduce noise
- Material support: Roughness, IOR, Specular, Metallic and Transparent
- Supports loading in '.vox' and '.bin' assets
- Ray traced Spheres

The Snake game is integrated into the renderer and contains smooth grid movement, background music, sound effects, visual effects and a cinematic camera spline which showcases the game.

I once again used ImGui to create a debug menu that allows real time interaction with the scene. This includes adding / modifying / deleting voxels, lights and spheres, as well as tweaking rendering and gameplay variables.
