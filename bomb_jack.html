Bomb Jack

Bomb Jack is a small retro arcade game originally released in 1984.
I started this project to improve my C++ skills and recreating Bomb Jack seemed like a fun challenge.

The goal of the game is to collect all the bombs in each level while avoiding the enemies, you lose the game if you lost all 3 lives.

As you progress, the levels of the game become increasingly more difficult with more dangerous enemies and level layouts.
The game features 5 exciting levels, background music, sound effects and responsive controls.

I completed this project within 2 weeks and enjoyed working on it a lot.
